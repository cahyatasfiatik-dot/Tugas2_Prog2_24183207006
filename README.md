
##Class Mahasiswa
Menyimpan data mahasiswa: **npm, nama, prodi**.
Memiliki **konstruktor**, **setter & getter**, dan method **tampilkanData()** untuk menampilkan data ke layar.

##Class MainApp
Program utama yang menampilkan **menu**:
  1. Tambah Data
  2. Tampilkan Data
  3. Keluar
Jika pilih **Tambah Data**, program membuat objek `Mahasiswa` dengan data yang dimasukkan.
Jika pilih **Tampilkan Data**, program menampilkan data mahasiswa yang tersimpan.
Program berjalan berulang sampai user memilih keluar
